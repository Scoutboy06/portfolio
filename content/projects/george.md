A calculation program written in Microsoft Excel to calculate the skin friction on various types of piles. The program is developed for use by geotechnical engineers in the design of pile foundations. The purpose of the program is to facilitate and streamline the calculations of skin friction, which is an important part of geotechnical design and analysis. The program is designed to be user-friendly and intuitive, with clear instructions and input fields for various parameters that affect skin friction. It includes features to handle different pile types, soil layers, and loading conditions, allowing users to customize the calculations according to specific project needs.
